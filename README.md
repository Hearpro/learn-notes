# cc-cli

`cc-cli` is a command-line tool for installing development templates.

## Installation

To install `cc-cli`, use the following command:

```shell
pnpm i -g @cc-heart/cc-cli
```

## Usage

create a repo from template

```shell
pnpm cc-cli
```
